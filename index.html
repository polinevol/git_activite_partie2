A la découverte de Git

Qu'est-ce que Git ?
Git est un outil de versionning permettant d'avoir des traces sur toutes les modifications faites sur votre code. Il facile la sauvegarde de vos sources et le travail collaboratif entre développeurs


Les principales commande à retenir
  • La commande "init" : elle sert à initialiser un répertoire git; à partir de ce moment git a un regard attentif sur tout ce qui se passe dans ce répertoire
  • La commande "add" : elle sert à ajouter un fichier dans le repository de git
  • La commande "commit" : elle sert à ajouter un point de sauvegarde de l'état de son repository git
